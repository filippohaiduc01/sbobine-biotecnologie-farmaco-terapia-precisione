# Contribuire / Contributing

## 🇮🇹 Italiano

Al momento questa repository è mantenuta da un singolo studente e **non accetta pull request esterne**. Questo potrebbe cambiare in futuro.

Se hai:
- una correzione da segnalare (errore di trascrizione, contenuto impreciso),
- una richiesta di rimozione contenuti,
- un suggerimento sulla struttura,

apri pure una **Issue** su GitHub: verrà letta e gestita dal maintainer.

---

## English

This repository is currently maintained by a single student and **does not accept external pull requests** at this time. This may change in the future.

If you have:
- a correction to report (transcription error, inaccurate content),
- a content removal request,
- a suggestion about the structure,

feel free to open an **Issue** on GitHub — it will be reviewed by the maintainer.
